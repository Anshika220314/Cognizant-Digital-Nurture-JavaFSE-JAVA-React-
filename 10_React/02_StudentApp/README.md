# 2. ReactJS-HOL — Student Management Portal (Class Components)

**Goal:** Create a class component named `Home`, plus `About` and
`Contact`, each displaying a welcome message, and render all three from
`App.js`.

## Structure
- `src/Components/Home.js` — class component, `Component`/`render()`
- `src/Components/About.js` — class component
- `src/Components/Contact.js` — class component
- `src/App.js` — renders `<Home/>`, `<About/>`, `<Contact/>`

## Run it
```bash
npm install
npm start
```
