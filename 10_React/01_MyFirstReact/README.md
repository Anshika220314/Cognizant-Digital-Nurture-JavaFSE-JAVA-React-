# 1. ReactJS-HOL — My First React App

**Goal:** Set up a React environment with `create-react-app` and print a
welcome heading.

## What this does
`App.js` renders a single `<h1>` heading: *"Welcome the first session of
React"*.

## Run it
```bash
npm install
npm start
```
Open http://localhost:3000 — you'll see the heading rendered on the page.
